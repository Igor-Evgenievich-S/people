MAG-RIFF

Адаптивная электрогитара из полимерного волокна PETG CF. Инновационный дизайн, передовые технологии.

О проекте

MAG-RIFF - это революционная адаптивная электрогитара, полностью состоящая из полимерного волокна PETG CF. Этот материал значительно прочнее традиционного пластика PLA, использовавшегося в предыдущих версиях.

Гитара состоит из трех основных частей, соединенных неодимовыми магнитами с дополнительной фиксацией через специальные пазы, предотвращающие смещение деталей.

Встроенный динамик позволяет играть без комбоусилителя, а уникальная система крепления педали эффектов открывает новые возможности для музыкантов.

Ключевые особенности

🔋

Встроенный аккумулятор

Литий-ионный аккумулятор с зарядкой через USB Type-C. Не требует снятия задней крышки для подзарядки.

🔊

Встроенный динамик

Динамик расположен в поддержке для руки, позволяет играть без дополнительного оборудования.

🧲

Магнитное соединение

Все детали скрепляются неодимовыми магнитами с дополнительной фиксацией через пазы.

🎛️

Модульная педаль эффектов

Педаль эффектов легко вставляется и извлекается благодаря предпружинным контактам.

🏗️

Безголовый дизайн

Все колки расположены в бридже, что улучшает баланс и эргономику инструмента.

🎸

Мощный хамбакер

Бриджевый хамбакер обеспечивает мощное звучание для тяжелого рока и других жанров.

Конструкция гитары

1

Основная часть

Состоит из грифа и деки, в которой расположена большая часть деталей. Изготовлена из прочного полимерного волокна PETG CF.

2

Поддержка для руки

Содержит встроенный динамик и механизм авторемня, принцип работы которого позаимствован у обычной рулетки.

3

Поддержка для ноги

В этой детали находится педаль эффектов. Использованы предпружинные контакты для легкого извлечения и установки педали.
